Project Title:

Air Quality Analysis Dashboard - India

Objectives:

Analyze air quality data collected from monitoring stations across India.
Compare the average AQI levels across different states and pollutants.
Identify pollutant-wise air quality patterns.
Create an interactive dashboard for easy data exploration and decision-making.

Dataset Information:

Dataset: Air Quality of India from Various Monitoring Locations
Country: India
Records: 505 Monitoring Stations
States Covered: 29
Cities Covered: 267
Attributes: State, City, Station, Latitude, Longitude, Pollutant ID, Pollutant Minimum, Pollutant Maximum, Pollutant Average, Last Update.

Tools & Technologies Used:

Microsoft Excel (Data Cleaning)
Power BI Desktop (Dashboard Development)

Data Cleaning & Transformation:

Removed duplicate records.
Checked and handled missing values.
Corrected data types.
Standardized column names.
Verified state and city names.
Loaded the cleaned dataset into Power BI.
Created KPI cards, slicers, charts, and map visuals.

Dashboard Overview:

The dashboard includes:

KPI Cards showing Total States (29), Total Cities (267), Total Stations (505), and Average AQI (33.2).
State, City, and Pollutant slicers for interactive filtering.
AQI monitoring data table.
Donut chart showing average AQI by pollutant.
Map showing monitoring station locations.
Bar chart showing average AQI by state.
Bar chart comparing average AQI by pollutant.

Key Insights:

PM10 recorded the highest average AQI (83.41), indicating it is the most significant pollutant in the dataset.
NH3 recorded the lowest average AQI (5.24), showing minimal contribution compared to other pollutants.
Delhi has the highest average AQI (54.44), indicating poorer air quality than other states in the dataset.
The dashboard covers 29 states, 267 cities, and 505 monitoring stations across India.

This dashboard provides a clear and interactive overview of India's air quality data. The project demonstrates how Power BI can transform raw environmental data into meaningful insights through interactive visualizations.
